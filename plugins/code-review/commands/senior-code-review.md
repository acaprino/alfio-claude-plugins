# Senior Code Review

Use the `senior-code-reviewer` agent to perform a comprehensive code review.

**Usage:** Provide file paths, directories, or a description of what to review. Optionally specify a focus area.

**Examples:**
- `src/api/auth.py` — review a specific file
- `src/payments/` — review a directory
- `security: src/api/` — focus review on security concerns
- `performance: src/db/queries.py` — focus review on performance

**Focus areas:** security, performance, architecture, testing, maintainability
